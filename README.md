# Oneulbread

## 프로젝트 소개

‘오늘빵’ 이라는 가상의 베이커리 브랜드를 설정하고 제품 예약, 픽업, 베이킹 클래스 예약 서비스를 제공합니다.

HTML, CSS, JS 를 이용한 UI 구현과 MVC패턴을 이해하고 DataBase와 연동하여 실제 웹사이트와 유사한 동작을 재연해 보기 위해 기획 하였습니다.
